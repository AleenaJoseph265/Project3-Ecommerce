# 📊 Sales Trends & Interaction Analysis

## 📌 Project Overview
This project analyzes user behavior and sales performance in an e-commerce platform. It focuses on conversion rates, revenue trends, and customer interaction patterns to identify business opportunities.

---

## 🎯 Objectives
- Analyze total sales and revenue trends
- Understand user conversion funnel
- Identify cart abandonment issues
- Find top-performing brands

---

## 🛠 Tools Used
- Power BI  
- SQL  
 

---

## 📊 Key Metrics
- **Total Revenue:** 6.30M  
- **Total Purchases:** 19.06K  
- **Total Views:** 139.82K  
- **Cart Abandonment Rate:** 77.23%  

---

## 🔄 Process
1. Data Collection from e-commerce dataset  
2. Data Cleaning & Transformation (ETL)  
3. Data Modeling in Power BI  
4. Dashboard Creation & Visualization  

---

## 📈 Key Insights
- High **cart abandonment rate (77%)** indicates user drop-off before purchase  
- Conversion funnel shows only **13.6% users complete purchase**  
- **November** recorded highest sales, while **December dropped significantly**  
- Top brand (**Runail**) generates highest revenue  
- Sales trend shows seasonal variation  

---

## 📊 Dashboard Features
- KPI Cards (Revenue, Purchases, Views)  
- Monthly Sales Analysis  
- User Conversion Funnel  
- Top Brands by Revenue  
- Sales Trend Over Time  

---

## 🚀 Conclusion
This analysis helps identify critical drop-off points in the user journey and highlights opportunities to improve conversion rates and revenue.
